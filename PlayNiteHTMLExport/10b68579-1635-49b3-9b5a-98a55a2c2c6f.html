Description

Cotton Reboot! is a completely remastered version of Cotton.Fantastic Night Dreams, the originator of the witchy shooting game released 30 years ago, with new features and the latest visual design.
The game also features 3 game modes.
  • X68000 original mode
faithfully emulates Sharp’s personal computer from the 1993 version.
  • Arrange mode
features 16:9 screens, redesigned graphics and characters.
  • Time attack mode
the player competes for scores online within a limited time.(2 minute and 5 minute mode can be selected)

Cotton reboot!　Ver 1.01 update patch is now available.


1. controller button notation
　Xbox controllers and other controllers (general Xinput-compatible controllers and DirectX-compatible controllers)
　When connected, the button notations have been modified to conform to the Xbox controller specifications.
　PS4 controller is still used when connected to a PS4 controller.
　If a PS4 controller is not connected, the buttons will be displayed as Xbox buttons.

2. Addition of the game exit menu
　Added to the main menu.

3. Screen mode change menu added
　An item has been added to the config to allow selection of full screen mode or windowed mode.
　If you change the mode and save the configuration, the next time you start up the system, the screen mode will be the one selected at the time of saving.

4. speed adjustment for monitors with a refresh rate of 60 Hz or higher
　Changed speed management to be independent of refresh rate when in window mode.
